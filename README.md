# py06

このフォルダでは、Python のモジュール分割とパッケージ設計を学びます。
関連する処理をまとめてパッケージ化したり、import の関係を整理したりしながら、複数ファイル構成の実装に慣れます。

## 何をする課題か

- `alchemy/`: 関連する処理をまとめて、パッケージとして扱う
- `ft_circular_curse.py`: モジュール同士が互いに依存しすぎないように整理する
- `ft_import_transmutation.py`: import の関係を見直して、読みやすい構成にする
- `ft_pathway_debate.py`: 複数モジュールに役割を分けて連携させる
- `ft_sacred_scroll.py`: 使い回しやすい形にコードを分割する

## ここで学ぶこと

- パッケージとモジュールを正しく構成する
- 循環依存や import の問題に対応する
- 役割ごとにコードを分割して整理する

## 実行方法

1. `cd py06`
2. `python3 ft_*.py`

`py06` は、複数ファイル構成での実装を経験するステージです。

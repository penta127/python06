# py06

Python のモジュール分割とパッケージ設計を学ぶ演習フォルダです。
特に循環インポートや依存関係を整理する方法に焦点があります。

## どんなことをやるか

- `alchemy/`: パッケージとして構成されたモジュール群
- `ft_circular_curse.py`: 循環依存を避けるための設計
- `ft_import_transmutation.py`: 遅延インポートや依存関係の扱い
- `ft_pathway_debate.py`: モジュール間の連携と役割分担
- `ft_sacred_scroll.py`: パッケージ構成での再利用性の検証

## 目的

- パッケージとモジュールを正しく構成する力をつける
- 循環依存や import の問題に対応する方法を学ぶ
- 役割ごとにコードを分割して整理する

## 実行方法

1. `cd py06`
2. `python3 ft_*.py`

`py06` は、複数ファイル構成での実装を経験するステージです。
